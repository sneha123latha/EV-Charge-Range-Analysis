# Electric Vehicle Charge and Range Analysis

## Project Overview
This project analyzes Electric Vehicle (EV) charging infrastructure and driving range using data visualization techniques.  
The goal is to understand EV adoption trends, charging station availability, and vehicle performance.

## Objectives
- Analyze EV charging stations distribution
- Compare EV range performance
- Identify affordable electric cars
- Study EV adoption trends in India
- Visualize insights using dashboards

## Tools & Technologies
- Tableau
- Excel
- CSV Datasets
- Data Visualization
- Data Cleaning

## Dataset Used
- EV India dataset
- Electric car specifications dataset
- Charging station dataset

## Features
- Interactive Dashboard
- EV Range Comparison
- Charging Station Analysis
- Cheapest Electric Cars Identification

## Project Output
The dashboard provides insights about:
- Range vs Price
- Charging station availability
- Vehicle performance
- Market trends

## Conclusion
Electric Vehicles are rapidly growing in India. Proper charging infrastructure and affordable pricing play a key role in adoption. This analysis helps understand current trends and future scope.
